# Business Intelligence project
 
## Project Title
Multi-Platform E-Commerce Analytics: Sales, Pricing, and Customer Behavior

## Project Overview
This project is part of the IT300 Business Intelligence course.
It focuses on analyzing e-commerce sales data from multiple platforms (Amazon, Jumia, and Souq)
to understand sales performance, pricing strategies, and customer behavior.

## Dataset
- Source: Kaggle, E-commerce sales dataset (Amazon, Jumia, Souq)
- Rows: 10,000+
- Format: CSV

## Project Structure
- /data_raw: Raw datasets
- /data_cleaned: Cleaned datasets after ETL
- /etl: Data cleaning and transformation scripts
- /model: Data model and star schema
- /dashboard: Power BI dashboards
- /report: Project reports (Week 1, Insights)
- /presentation: Final presentation slides
